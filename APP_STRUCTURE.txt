Key Features Workflow:
=====================

1. Voice-to-Category Flow:
   User speaks → Web Speech API → LLM categorizes → Form auto-filled

2. Receipt + Voice Flow:
   Upload receipt → View reference → Speak details → AI categorizes

3. Predictive Alerts Flow:
   Add expenses → Click Insights → ML analyzes → Shows forecasts

4. Multi-Currency Flow:
   Type description → AI detects currency → Auto-selects

5. Export Flow:
   Click Export → Choose format → Download/Print

Data Storage:
============
Browser localStorage:
├─ vaultspend_expenses (Expense[])
├─ vaultspend_categories (Category[])
├─ vaultspend_budgets (Budget[])
├─ vaultspend_alerts (PredictiveAlert[])
└─ vaultspend_settings (AppSettings)

OPFS (Origin Private File System):
├─ LFM2-350M model (~250MB)
└─ Other cached models

Tech Stack:
==========
Frontend: React 19 + TypeScript + Vite
AI SDK: RunAnywhere Web SDK
Models:
  - LFM2 350M Q4_K_M (Language)
  - Web Speech API (Voice input)
Storage: localStorage + OPFS
Deployment: Vercel (with COOP/COEP headers)
